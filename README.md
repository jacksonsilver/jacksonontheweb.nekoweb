# Jackson On The Web - React App

This is a React conversion of the Jackson On The Web static website.

## Setup

Install dependencies:
```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

Build for production:
```bash
npm run build
```

The compiled files will be output to `jacksonontheweb.nekoweb.org/dist/` and are ready to deploy to Nekoweb.

## Deployment to Nekoweb

1. Run `npm run build` to build the React app
2. The files in `jacksonontheweb.nekoweb.org/` are all that's needed for the website
3. Upload the contents of `jacksonontheweb.nekoweb.org/` to your Nekoweb site
4. The app will be accessible at `https://jacksonsilver.nekoweb.org/`

## File Structure

- `index.html` - Entry point for the React app
- `dist/` - Compiled React app (generated by `npm run build`)
- `assets/` - Static assets (logo, images)
- `.htaccess` - Server configuration for proper routing

## Preview

Preview the production build locally:
```bash
npm run preview
```
